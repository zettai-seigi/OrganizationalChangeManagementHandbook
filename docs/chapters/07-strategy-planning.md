# Chapter 7: OCM Strategy and Planning

## Learning Objectives

After completing this chapter, you will be able to:
- Develop comprehensive OCM strategies
- Create integrated OCM plans
- Align OCM with project timelines
- Define resource requirements and governance

---

## OCM Strategy Development

### Strategy Components

| Component | Description |
|-----------|-------------|
| **Approach** | Overall methodology (e.g., ADKAR-based) |
| **Scope** | Boundaries of OCM effort |
| **Governance** | Decision-making and oversight |
| **Resources** | Team, budget, tools |
| **Timeline** | Phases and milestones |
| **Success Criteria** | How success will be measured |

---

## OCM Planning Framework

### Core Plans

1. **Communication Plan**: Who, what, when, how
2. **Training Plan**: Learning strategy and logistics
3. **Sponsorship Plan**: Sponsor engagement activities
4. **Resistance Plan**: Anticipation and response strategies
5. **Reinforcement Plan**: Sustainability mechanisms

---

## Key Takeaways

- OCM strategy must align with overall project approach
- Planning should be comprehensive but flexible
- Resource and governance structures enable execution
- Plans must integrate with project milestones
