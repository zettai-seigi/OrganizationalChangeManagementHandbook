# Chapter 8: Communication Planning and Execution

## Learning Objectives

After completing this chapter, you will be able to:
- Design comprehensive communication plans
- Craft effective change messages
- Select appropriate channels for different audiences
- Measure communication effectiveness

---

## Communication Planning

### The 5 Rights of Change Communication

| Right | Question |
|-------|----------|
| **Right Message** | What do they need to know? |
| **Right Audience** | Who needs this information? |
| **Right Time** | When should they receive it? |
| **Right Channel** | How should it be delivered? |
| **Right Sender** | Who should deliver it? |

---

## Message Framework

### Key Messages by Audience

| Audience | Key Messages |
|----------|-------------|
| **Executives** | Strategic alignment, ROI, risks |
| **Managers** | Team impacts, how to support |
| **End Users** | What's changing, WIIFM, timeline |
| **Support Staff** | Technical details, support expectations |

---

## Communication Channels

| Channel | Best For | Frequency |
|---------|----------|-----------|
| Town Halls | Major announcements, Q&A | Monthly |
| Email | Updates, documentation | As needed |
| Intranet | Reference information | Continuous |
| Team Meetings | Local discussion | Weekly |
| 1:1 Conversations | Individual concerns | As needed |

---

## Key Takeaways

- Communication must be planned, not ad-hoc
- Different audiences need different messages
- Two-way communication enables feedback
- Repetition is necessary for message retention
