## Visual ID: 12-Figure-12.1
**Type**: Dashboard Infographic
**Title**: Six Key Performance Indicators Dashboard
**Chapter Reference**: Chapter 12: Adoption Monitoring and Measurement

### Purpose Description
Provides a visual dashboard representation of the six core OCM KPIs with their targets and typical measurement ranges. This serves as the primary reference for stakeholders to understand OCM measurement framework at a glance.

### Content Description
**Structure**:
- Grid layout with 6 KPI cards arranged in 2 rows of 3
- Each card contains: KPI name, definition, target value, gauge/meter visual, and status indicator
- Consistent card design across all six metrics
- Color-coded status indicators (green/yellow/red)

**Visual Flow**:
- Left to right, top to bottom reading pattern
- Top row: Awareness Level, Adoption Rate, Proficiency Level
- Bottom row: Resistance Level, Sponsor Engagement Score, Sustainment Rate

**Key Elements**:
1. **Awareness Level Card**:
   - Circular gauge showing 0-100%
   - Target line at 100%
   - Definition: "% of stakeholders aware of change"
   - Icon: lightbulb or eye symbol

2. **Adoption Rate Card**:
   - Circular gauge showing 0-100%
   - Target line at 85%
   - Definition: "% of users actively using new processes"
   - Icon: checkmark in circle

3. **Proficiency Level Card**:
   - Circular gauge showing 0-100%
   - Target line at 80%
   - Definition: "% of users meeting competency standards"
   - Icon: trophy or star symbol

4. **Resistance Level Card**:
   - Circular gauge showing 0-100%
   - Target line at 10% (inverted - lower is better)
   - Definition: "% showing active resistance"
   - Icon: alert or barrier symbol

5. **Sponsor Engagement Score Card**:
   - Horizontal bar or star rating
   - Definition: "Measure of sponsor activity and effectiveness"
   - Icon: person with crown or leadership symbol

6. **Sustainment Rate Card**:
   - Circular gauge showing 0-100%
   - Target line at 90%
   - Definition: "% of adoption maintained over time"
   - Icon: anchor or infinity symbol

### Specifications
- **Dimensions**: 1200px × 800px (landscape orientation)
- **Card Size**: Each card approximately 350px × 350px
- **Colors**:
  - Card background: #F8F9FA (Light Gray)
  - Card borders: #6C757D (Medium Gray)

  - Target lines: #0066CC (Primary Blue)
  - Text: #333333 (Dark Gray)
- **Fonts**:
  - KPI Names: 18pt Bold, Sans-serif
  - Target Values: 24pt Bold, Sans-serif
  - Definitions: 12pt Regular, Sans-serif
- **Tools**: Designed in Figma, Adobe Illustrator, or PowerPoint

### Caption
**Figure 12.1: OCM Six Key Performance Indicators Dashboard** - The six KPIs provide a comprehensive view of adoption progress from initial awareness through sustained adoption. Each metric has specific targets based on research and best practices.

## Visual ID: 12-Figure-12.3
**Type**: Data Visualization Matrix
**Title**: Data Collection Methods Comparison Matrix
**Chapter Reference**: Chapter 12: Adoption Monitoring and Measurement

### Purpose Description
Provides a comprehensive comparison of four primary data collection methods, helping practitioners select appropriate approaches based on change characteristics and needs.

### Content Description
**Structure**:
- 4×5 comparison matrix
- Rows: System Data, Surveys, Observation, Manager Feedback
- Columns: What It Measures, Advantages, Limitations, Best Use Cases, Frequency

**Visual Flow**:
- Header row with column titles
- Left column with data source types
- Color-coded rows for visual differentiation
- Icons for each data source type

**Key Elements**:

1. **System Data & Analytics Row**:
   - Icon: computer with chart
   - What It Measures: Usage, transactions, errors, logins
   - Advantages: Objective, comprehensive, no burden
   - Limitations: Only system interaction, not quality/attitude
   - Best Use Cases: Technology changes, system adoption
   - Frequency: Daily to weekly

2. **Surveys & Pulse Checks Row**:
   - Icon: document with checkboxes
   - What It Measures: Sentiment, self-reported adoption, confidence
   - Advantages: Direct feedback, scalable, tracks trends
   - Limitations: Response bias, survey fatigue
   - Best Use Cases: Awareness, desire, perception
   - Frequency: Bi-weekly to monthly

3. **Observation & Spot-Checks Row**:
   - Icon: eye or binoculars
   - What It Measures: Actual behaviors, compliance, workarounds
   - Advantages: Real behavior, identifies issues
   - Limitations: Time-intensive, small sample
   - Best Use Cases: Process changes, quality verification
   - Frequency: Weekly to monthly

4. **Manager & Change Agent Feedback Row**:
   - Icon: people discussing
   - What It Measures: Team status, concerns, resistance, wins
   - Advantages: Ground-level intelligence, contextual
   - Limitations: Subjective, may be filtered
   - Best Use Cases: Issue identification, early warnings
   - Frequency: Weekly

### Specifications
- **Dimensions**: 1600px × 700px
- **Cell Padding**: 15px all sides
- **Colors**:
  - Header row: #0066CC (Primary Blue), white text
  - Row 1 (System Data): Light blue tint #E6F2FF
  - Row 2 (Surveys): Light green tint #E6F9E6
  - Row 3 (Observation): Light yellow tint #FFF9E6
  - Row 4 (Manager Feedback): Light purple tint #F2E6FF
  - Borders: #6C757D (Medium Gray)
  - Text: #333333 (Dark Gray)
- **Fonts**:
  - Headers: 14pt Bold, Sans-serif
  - Content: 11pt Regular, Sans-serif
- **Tools**: Excel, Google Sheets, or custom HTML/CSS table

### Caption
**Figure 12.3: Data Collection Methods Comparison** - Multiple data sources provide a complete picture of adoption. Each method has strengths and limitations; effective measurement combines multiple approaches.

### Position in Chapter
Insert at the beginning of "Data Collection Methods" section, before detailed subsections.

### Notes

- Ensure table is responsive for digital viewing
- Consider expandable rows for more detail in digital version
