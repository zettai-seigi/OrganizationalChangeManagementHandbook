## Key Features Across All Visuals

### Design Principles
- **Consistency**: Uniform color palette, typography, and iconography
- **Accessibility**: Color-blind friendly with patterns/icons, high contrast
- **Scalability**: Designed for both digital viewing and print
- **Clarity**: Clear hierarchies, readable at various sizes
- **Actionability**: Practical, implementable guidance

### Standard Elements
- **Icons**: Consistent 24px, 32px, or 48px sizes
- **Typography**:
  - Titles: 16-20pt Bold
  - Headings: 13-16pt Bold
  - Body: 10-11pt Regular
  - Notes: 9-10pt Italic
- **Borders**: 1-3px, color-coded by category
- **Spacing**: Consistent 20-50px margins and padding

### Interactive Features (Digital)
- Expandable sections for additional detail
- Drill-down capability on complex visuals
- Hover states for tooltips and definitions
- Clickable elements linking to related content
- Animated versions showing progressive disclosure

### Print Considerations
- All visuals printable in grayscale while remaining legible
- Page size compatibility (Letter, A4)
- High resolution (300 DPI minimum)
- Clear labels and legends
