# Organizational Change Management Handbook

A Comprehensive Guide to ITIL/ITSM Organizational Change Management Best Practices

## Overview

This handbook provides a complete guide to Organizational Change Management (OCM) within IT Service Management environments. It covers the frameworks, methodologies, tools, and techniques needed to successfully manage the people side of IT transformations.

## Live Site

## Key Framework Elements

### The ADKAR Model
- **A**wareness - Understanding why change is needed
- **D**esire - Personal motivation to support change
- **K**nowledge - Understanding how to change
- **A**bility - Skills and behaviors to implement change
- **R**einforcement - Sustaining change over time

### 8 Critical Success Factors
1. Executive Sponsorship and Commitment
2. Clear Vision and Case for Change
3. Stakeholder Engagement and Involvement
4. Effective Communication
5. Training and Capability Building
6. Change Agent Network
7. Resistance Management
8. Measurement and Reinforcement

### 6 Key Performance Indicators
1. Awareness Level (target: 100%)
2. Adoption Rate (target: ≥85%)
3. Proficiency Level (target: ≥80%)
4. Resistance Level (target: ≤10%)
5. Sponsor Engagement Score
6. Sustainment Rate (target: ≥90%)

## Local Development

```bash
# Clone the repository
git clone https://github.com/zettai-seigi/OrganizationalChangeManagementHandbook.git

# Navigate to docs directory
cd OrganizationalChangeManagementHandbook/docs

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# View at http://localhost:4000/OrganizationalChangeManagementHandbook/
```
